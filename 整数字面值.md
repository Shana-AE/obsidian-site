| Number Literrals | Example     |
| ---------------- | ----------- |
| Decimal          | 98_222      |
| Hex              | 0xff        |
| Octal            | 0o77        |
| Binary           | 0b1111_0000 |
| Byte(u8 only)    | b'A'        |

-   除了byte类型外，所有的数值字面值都允许使用类型后缀
    -   例如57u8
-   如果不清楚应该使用哪种类型，可以使用Rust相应的默认类型
-   整数的默认类型是i32:
    -   总体上来说速度很快，即使在64位系统中
